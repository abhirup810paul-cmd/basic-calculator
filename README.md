# Simple Python Calculator

A lightweight, terminal-based calculator written in Python. This project serves as a clean, easy-to-understand implementation of basic arithmetic operations, making it perfect for beginners learning Python or Git version control.

---

## 🚀 Features

*   **Addition (`+`):** Sum two numbers.
*   **Subtraction (`-`):** Find the difference between two numbers.
*   **Multiplication (`*`):** Multiply two numbers.
*   **Division (`/`):** Divide one number by another (with built-in protection against division by zero).
*   **User-Friendly Interface:** Loop-based terminal menu that stays open until you choose to exit.

---
